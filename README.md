# shop_check
基于深度学习的【商品识别】系统设计与实现~Python

## 一、介绍
商品识别系统，本系统使用Python作为主要开发语言，通过收集了7种常见的商品图像数据集（'杯子', '椅子', '汽车', '电脑', '电视机', '衣服', '鞋子'），然后基于TensorFlow搭建卷积神经网络算法模型，然后进行多轮迭代训练，最后得到一个识别精度较高的模型文件，再使用Django开发了一个Web操作的网页平台，实现用户上传一张商品图片识别其名称。
